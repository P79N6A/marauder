# Contributing to Marauder

## commit 风格指南

commit 信息需遵循以下风格约定：

```
Tag(scope): Short description

Longer description here if necessary
```

commit 信息需要指定 **tag** 描述，可选值为：

- **build**: 项目构建系统相关调整
- **docs**: 修改文档
- **chore**: 内部调整（重构，添加测试，更新依赖...）
- **feat**: 新增功能
- **fix**: 修复 bug
- **perf**: 优化

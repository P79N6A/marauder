# Marauder

> 挖掘机技术哪家强

webpack-marauder2.0 下一代工程化体系

## Quick Overview

```sh
npx @mara my-app
cd my-app && npm dev
```

## 安装

```sh
yarn add @mara/cli -g
```

### 新建项目

via npm:

```sh
# or npm init @mara my-app
npx init @mara my-app
```

via yarn:

```sh
yarn create @mara my-app
```

via @mara/cli:

```sh
mara create my-app
```

## Roadmap

- **@mara/cli** 全新 CLI 工具
